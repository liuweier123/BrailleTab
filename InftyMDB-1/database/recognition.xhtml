Mathematical Formula Recognition Database

Overview

This database was constructed by collecting recognition results generated by InftyReader. It contains 3,000 correctly recognized mathematical formulae and 1,400 misrecognized mathematical formulae collected from 32 pure mathematical articles.


CSV files of Recognition Results

The following links are CSV files of a collection of recognition results of mathematical formulae generated by InftyReader. The CSV files consist of recognition results starting with a "Formula" line followed by a series of "Chardata" lines. A "Formula" line consists of the string "Formula", formula number (integer), and recognition status ("CorrectRecognition" or "Misrecognition"). A "Chardata" line consists of the string "Chardata", character ID (integer), coordinates of rectangular area (4 integers), math-flag (always "1"), link type to parental character (integer), the ID of parental character (integer), character name (string), character category (string), and internal character code of InftyReader (hexadecimal integer).
